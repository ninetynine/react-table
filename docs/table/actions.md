<br />

The `actions` prop is used to be appended on the end of each row when using the default implementation of the [`rowRenderer`](./row-renderer.md). This expects a valid React element.

```jsx
<Table
  actions={(
    <a>
      Click Here
    </a>
  )}
/>
```
